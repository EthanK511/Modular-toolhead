# Overview

## What we’re building
A modular quick-swap toolhead system for an Ender 3 V2 (Sprite direct drive already installed) that supports:
- normal printing
- a pen/marker attachment (plotting)
- optional light-duty cutting attachment (TBD, safety-first)

## Non-stock hardware changes (Rework validity)
- custom backplate / interface
- custom quick-swap attachment geometry (magnets + alignment)
- tool modules (pen, cutter)
- paper hold-down method for the build plate

## Key constraints
- Pen up/down via Z moves only (no servo)
- Must be designed by me (CAD + build)
- Avoid hazardous mods
